After six months of work, thirteen students have developed this application for mobile phone. Since september, the students spent two hours a week learning web programming languages : HTML, CSS and JavaScript. Thus, they have been able to create a hybrid multiplatform application ( Android , iOS , Windows Phone, BlackBerry ... ) based on the open-source framework Apache Cordova.

The application is still in "beta" stage.

Features :

  • access to the newsfeed of the French International School "Victor Segalen" of Hong Kong
  • access to Pronote
  • access to the FIS website
