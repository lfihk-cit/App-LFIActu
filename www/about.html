Après six mois de travail, treize élèves de classe de seconde, ont mis au point cette application pour téléphone mobile. Son développement a été réalisé dans le cadre de l'enseignement d'exploration : "Création et Innovation Technologiques".

Depuis septembre, les élèves ont consacré deux heures par semaine à l'apprentissage des langages de programmation web : le HTML, le CSS et le JavaScript. Ces connaissances leur ont permis de créer une application hybride multiplateforme (Android, IOS, Windows Phone, BlackBerry...) s'appuyant sur le framework open-source Apache Cordova.

Toujours en version "beta", l'application permet:

  • de lire le fil d'actualité du Lycée Français International "Victor Segalen" de Hong Kong
  • d'accéder directement à Pronote
  • d'accéder directement au site internet du LFI
